FM Index of a DNA sequence contained in a FASTA file

Sara Resta

2023-08-28

library(FMIndex)

Introduction

FM index is a very important tool for compressing data

FM index data structures

F column

The F column is given as a integer, the names of the elements are the characters of the sequence. In this example the argument includeEndChar is not specified so itâ€™s set to its default value TRUE.

getFcolumn(Biostrings::DNAString("ACCT"))
## . A C T 
## 1 1 2 1

This is an example with the argument argument includeEndChar set to FALSE.

getFcolumn(Biostrings::DNAString("ACCT"))
## . A C T 
## 1 1 2 1

L column

The L column is the Burrows-Wheeler transformation of the give sequence. The function returns also the suffix array.

getLcolumn(Biostrings::DNAString("AACCGT"))
## $bwt
## 7-letter DNAString object
## seq: T.AACCG
## 
## $suffix_array
## [1] 7 1 2 3 4 5 6

Tally table

The Tally table contains the occurrences of each unique character for each position of the L column (BWT). It can be compressed saving just some rows with regular intervals (checkpoints).

In this example each row of the Tally Table is saved

getTallyTable(Biostrings::DNAString("T.AACCG"))
##      A C G T
## [1,] 0 0 0 1
## [2,] 0 0 0 1
## [3,] 1 0 0 1
## [4,] 2 0 0 1
## [5,] 2 1 0 1
## [6,] 2 2 0 1
## [7,] 2 2 1 1

Here we save a checkpoint every two rows.

getTallyTable(Biostrings::DNAString("T.AACCG"),2)
##      A C G T
## [1,] 0 0 0 1
## [2,] 1 0 0 1
## [3,] 2 1 0 1
## [4,] 2 2 1 1

Suffix Array

getSA(Biostrings::DNAString("ACGTTC"))
## [1] 7 1 6 2 3 5 4

  • tallywidth=1

  • includeEndChar=TRUE
